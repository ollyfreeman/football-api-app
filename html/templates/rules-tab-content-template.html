Introduction

This game is designed to prototype a new gambling application that takes inspiration from classic British football 'Pools' betting. The final product would use live updates from football-api.com. However, to make designing, testing and demonstrating viable, it currently uses an API that was built to mock on-demand the live feed that would be received from football-api.com (see the API tab). This mock on-demand API has the option of simulating the matches at a higher rate - this game uses that functionality, so the games are simulated at x60 speed (i.e. 1 simulated minute lasts 1 second).

How to play

  1. Predict the outcome of each of the matches: to predict that either of the teams will win, you click on that team's name - to predict that the match will end in a draw, which is the default selection, you click the 'v' symbol.
  2. Click START.
  3. As the simulation proceeds, you can change your predictions. It is only your most recent prediction for each match that counts towards your score - but note that the later in the match that this prediction happens, the fewer points you receive if your prediction is correct (see 'Scoring' below). The graph displays your current score, which will change as goals are scored and as you change you predictions.

Scoring

The scoring system for the game is as follows:

  • The maximum possible score for all the matches combined is 100, the minimum score is 0.
  • If your prediction for a match is incorrect, you receive a score of 0 for that match.
  • If your prediction for a match is correct, you recieve a percentage of the maximum possible score for that match, depending on when that prediction was made.

For example, say Arsenal are playing against Manchester United and say that two thirds of the way through the match you predict that Arsenal will win. Therefore, if Arsenal go on to to win you will receive one third of the maximum possible points for that game. However, if Arsenal don't win then you receive 0 points for that game.

Match data

The simulated match data is from the Premier League matches played on the final day of the 2014-2015 season, on Sunday 24th May 2015.
